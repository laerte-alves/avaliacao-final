O QUE É GIT

Git é um sistema de controle de versão distribuído, usado principalmente no desenvolvimento de software, para rastrear mudanças em arquivos ao longo do tempo. Ele permite que várias pessoas trabalhem em um mesmo projeto de forma colaborativa, facilitando o gerenciamento de alterações e o trabalho em equipe.

Configuração Inicial

    
      git config --global user.name "Seu Nome"
      git config --global user.email "seuemail@exemplo.com"

Criar e Inicializar Repositório

    
      git init

Adicionar e Confirmar Alterações

    
      git add .
      git commit -m "Sua mensagem de commit"

Verificar o Status

    
      git status

Histórico de Commits

    
      git log

Branches

    
      git branch nome_da_branch
      git checkout nome_da_branch
      git merge nome_da_branch
      git branch

Trabalhando com Repositórios Remotos

    
      git remote add origin URL_do_repositorio_remoto
      git push origin nome_da_branch
      git pull origin nome_da_branch
      git clone URL_do_repositorio_remoto
